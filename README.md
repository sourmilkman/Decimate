# Decimate

An installable 3D destruction game built with Three.js, Rapier, TypeScript, and Vite.

## Run locally

```powershell
npm install
npm run dev
```

Open the displayed local URL in landscape orientation. Drag across the room and release to fire. Reach 75% decimation, then tap **Disguise** and select a glowing intact object before the timer expires.

## Verify

```powershell
npm test
npm run build
```
